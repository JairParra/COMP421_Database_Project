# indexes 
- Useful indexes for the database. 
- Examples are the first two of interesting queries under data_retrieve directory

## index1 
- create clustered index on height of database mate
- as shown in indexTest.log, scaning on mate with index takes only around half of the original time 

## index2 
- create clustered index on custName of database request
- as shown in indexTest.log, scaning on request with index takes only around half of the original time
